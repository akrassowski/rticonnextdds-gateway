Release Notes
=============

Supported Platforms
-------------------

|RTI_RSMQTT| is expected to build and run on the platforms in
:numref:`TableSupportedPlatforms`.

It can also be deployed as a library linked into your application.

This is true for all platforms in :numref:`TableSupportedPlatforms`.

.. list-table:: Supported Platforms
    :name: TableSupportedPlatforms
    :widths: 20 80
    :header-rows: 1

    * - Platform
      - Operating System

    * - Linux systems
      - All Linux platforms supported by |RTI_RS| and by |PAHO_ASYNC|.
    * - macOS systems
      - All macOS platforms supported by |RTI_RS| and by |PAHO_ASYNC|.
    * - Windows systems
      - All Windows platforms supported by |RTI_RS| and by |PAHO_ASYNC|.
